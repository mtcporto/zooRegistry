# **App Name**: Zoo Registry

## Core Features:

- Class Registration: Form for registering new animal classes (e.g., mammals, birds, reptiles).
- Order Registration: Form to register the order of an animal within its class.
- Family Registration: Form to register the family of the animal, which determines the order it belongs to.
- Animal Registration: Form to register a new animal with its scientific name, common name, and other names.
- Instance Registration: Form for registering individual animal instances, including details like registry number, origin, entry date, sex, and markings.
- Browse Registries: Browse the animal class, order, family, animal, and instance registries.

## Style Guidelines:

- Primary color: Soft blue (#7EC4CF) evoking a sense of tranquility and nature.
- Background color: Very light gray (#F5F5F5), nearly white, for a clean and modern feel.
- Accent color: Muted green (#98D8AA) to complement the blue and reinforce the natural theme, drawing focus to key actions.
- Clean and readable sans-serif font for all text elements.
- Card-based layouts for each animal, to maximize use of space on the screen.
- Simple, line-based icons for navigation and categories to maintain a minimalist design.
